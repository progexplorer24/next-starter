## Custom Font Size Values

In order to have more flexibility over which type scale to use (for more information visit [Type Sacle Section](/design-tokens/type-scale)).
To control font-size with custom classes use class `text-{number}`. This classes follow the same naming convention as spacing, so if you multiply number by four you will get the font-size in pixels.

The Only exception to this rule is `text-overline`, which is the smallest font-size available (10px). You may use it occasionally for annotations, image credits or for metadata of an articles.
